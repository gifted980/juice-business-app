# Juice Business App 🍹

This is a simple project for a small juice and snacks business.

## 📌 About the Project
The system helps display juice menu items and prices for customers.

## 🍊 Features
- Simple juice menu display
- Prices for each item
- Easy to expand into an ordering system

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
